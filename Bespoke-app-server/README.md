# Bespoke Dua API

## Setup

1. Copy `appsettings.Example.json` to `appsettings.json`.
2. Replace `YOUR_GEMINI_API_KEY_HERE` in `appsettings.json` with your [Google AI Studio](https://aistudio.google.com/app/apikey) API key.

`appsettings.json` is gitignored so your API key is never committed.
